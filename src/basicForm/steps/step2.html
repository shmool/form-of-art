We add a few elements to our form:

  • form tag (ngForm directive)
  • label for the input
  • 'id' for the input and 'for' attribute in the label tag pointing to the input's id (this way, clicking on the label sets focus on the input)
  • submit button - with ng-click binded to the submit() function in the controller - see basicFormController.js
  • the controller's alias 'demo' (we're using 'controllerAs' in the routing configuration - see app.js)
  • css classes which are defined in Bootstrap (we also wrap the label and the input with a div to add it a form-group class)

We can submit the form either by clicking on the submit button or by pressing Enter (since there's only one input field and one button with ngClick. See the HTML specifications for submission rules in the docs).
