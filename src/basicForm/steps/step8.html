To further demonstrate the use of ngModelOptions, we add two checkboxes.

  • Both checkboxes toggle the value of 'sendSpam', which is used as the value of ngShow for the "Spam will be sent" paragraph.
  • The First checkbox, labeled "Send lazy spam" does not contain any additional attributes.
  • The second checkbox, labeled "Send spam now" has the attribute ng-model-options="{updateOn: 'default'}". This overrides the form's ngModelOptions and updates the model immediately on change.
