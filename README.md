# Form of Art 

Demos for the lecture series. 
I used the [Angular + Webpack Starter (ES6)](https://github.com/nirkaufman/angular-webpack-starter.git)

### How to run:
1. clone the project
2. run `npm install`
3. run `npm start`
4. navigate to 'localhost/8080'
